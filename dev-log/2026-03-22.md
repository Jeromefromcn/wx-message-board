# Dev Log - 2026-03-22

## Summary

Initial creation and setup of the wx-message-board WeChat mini-program project. The full project structure was scaffolded, pushed to GitHub, and the user began importing it into WeChat Developer Tools. Cloud development environment was configured.

## Design Decisions

- **WeChat Cloud Development**: Chose WeChat's built-in cloud platform (wx.cloud) over a custom backend. This avoids server maintenance and leverages free tier for a small message board.
- **Two cloud functions**: `getMessages` (fetch latest 20 messages, ordered by createdAt desc) and `addMessage` (insert with openid, nickName, avatarUrl, content, createdAt). Kept minimal — no pagination or auth for now.
- **Branch strategy**: `develop` for active development, `main` for stable releases. Mirrors the blockly-platform convention.
- **WXML emoji**: Direct Unicode emoji characters instead of HTML entities — WXML does not support HTML entity encoding.
