# 開發日誌 - 2026-03-22

## 摘要

創建並初始化 wx-message-board 微信小程序項目。完成了完整的項目結構搭建、推送至 GitHub，用戶開始在微信開發者工具中導入項目，並完成了雲開發環境配置。

## 設計決策

- **使用微信云開發**：選擇微信內建雲平台（wx.cloud）而非自建後端，避免服務器維護，利用免費套餐滿足小型留言板需求。
- **兩個雲函數**：`getMessages`（獲取最新 20 條留言，按 createdAt 倒序）和 `addMessage`（插入含 openid、nickName、avatarUrl、content、createdAt 的留言）。保持最小化，暫不做分頁或身份驗證。
- **分支策略**：`develop` 用於開發，`main` 用於穩定版本，與 blockly-platform 保持一致。
- **WXML Emoji**：使用直接 Unicode 字符而非 HTML 實體——WXML 不支持 HTML 實體編碼。
